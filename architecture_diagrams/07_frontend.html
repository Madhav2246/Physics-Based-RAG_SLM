Frontend — frontend_new/

"Physics RAG · SLM Console". React 18 + TanStack Router (file-based routes) + shadcn/ui (Radix) + Tailwind, built with Vite. LaTeX rendering via react-katex (components/Latex.tsx).

Shell & plumbing

components/AppShell.tsx

  • Fixed 64-wide glass sidebar; nav: Q&A Playground (/), Knowledge Base (/knowledge), Evaluation (/evaluation), Model Tuning (/tuning).
  • API health pill — polls checkHealth() (GET /, 3 s timeout) every 15 s; shows "API: Online/Offline :8000".

lib/api.ts

  • API_BASE = "http://localhost:8000".
  • apiFetch<T>(path, init): fetch + AbortController timeout (default 35 s, per-call override), auto Content-Type: application/json unless FormData, throws on non-2xx.

Data-shape contract with the backend

Types in index.tsx mirror api_server serializers exactly: ExploreResult {success, target, symbolic, numeric, sanity_ok, error, corpus_equation, provenance[], provenance_fraction, sweep{x,y,…}} (from _serialize_explore_result) and RagMeta {mode, node_profile, explore_result, sweep_plot_url}. Confidence breakdown rows are {label, value, max, ok} from _format_breakdown — real engine values, not recomputed client-side.
