HOW TO USE (Q&A Playground)
  1. Click the MODE button shown in [BRACKETS] under the query box.
     (AUTO also works for every one of these — the button just
      guarantees the path if auto-detect ever wavers.)
  2. Paste the question exactly as written.
  3. Press Query.

WHAT EACH MODE SHOWS THE EVALUATOR
  LOOKUP   -> governing equation + symbolic/dimensional/numerical
              validators + a confidence score.
  EXPLORE  -> "Deterministic Design Synthesis" card: one solved
              numeric value (no LLM guessing — pure SymPy algebra).
  SWEEP    -> a "Parametric Sweep" plot (PNG) under the answer.


----------------------------------------------------------------
LOOKUP  —  explain the governing equation
----------------------------------------------------------------
[LOOKUP]  What is the drain current equation for a MOSFET in saturation?
          expect:  Id = 0.5 * Cox * (W/L) * mu * (Vgs - Vth)^2

[LOOKUP]  What is the subthreshold swing equation?
          expect:  SS = (kT/q) * ln(10) * (1 + Cd/Cox)

[LOOKUP]  What is the saturation drain current equation with channel length modulation?
          expect:  Id = 0.5 * Cox * mu * (W/L) * Vov^2 * (1 + lam * Vds)


----------------------------------------------------------------
EXPLORE  —  solve for a single numeric value
----------------------------------------------------------------
[EXPLORE]  Calculate the output resistance ro for Id = 100uA
           expect:  ro = 1 / (lam * Id)  =  1.0e5 ohm  (100 kohm)

[EXPLORE]  Find the transconductance gm for Vov = 0.4V
           expect:  gm = mu * Cox * (W/L) * Vov  =  4.0e-3 S

[EXPLORE]  Find the drain current Id for Vgs = 1.0V
           expect:  Id = 0.5 * Cox * mu * (W/L) * (Vgs - Vth)^2  =  1.8e-3 A


----------------------------------------------------------------
SWEEP  —  generate a parametric plot (PNG appears under the answer)
----------------------------------------------------------------
[SWEEP]  Plot ro versus Id from 10uA to 1mA
         expect:  decreasing curve, ro = 1/(lam*Id)     -> sweep_ro_vs_Id.png

[SWEEP]  Plot gm versus Vov from 0.1V to 0.6V
         expect:  straight line, gm = mu*Cox*(W/L)*Vov  -> sweep_gm_vs_Vov.png

[SWEEP]  Plot Id versus Vds from 0.1V to 2V
         expect:  Id rising with CLM slope (1+lam*Vds)  -> sweep_Id_vs_Vds.png

[SWEEP]  Plot Vth versus Vsb from 0V to 1V
         expect:  body-effect curve,
                  Vth = Vth0 + gamma*(sqrt(2*Phi_f+Vsb) - sqrt(2*Phi_f))
                                                          -> sweep_Vth_vs_Vsb.png


----------------------------------------------------------------

         expect:  classic MOSFET output characteristics — 3 Id-Vds curves,
                  one per overdrive Vov (0.3 / 0.5 / 0.7 V)
                                              -> sweep_Id_vs_Vds_Vov_family.png
